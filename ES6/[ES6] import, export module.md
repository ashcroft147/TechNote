## import
 - import name from 'module-name';
    - module-name: module명, 파일명

## module definition
export name1, name2, ..., nameN;
- name: export할 속성이나 함수, 객체

export *;
- *: 파일 내의 정의된 모든 속성, 함수, 객체

export default name;
- default: 모듈의 기본값. 파일 내에서 한 번만 호출될 수 있다.
