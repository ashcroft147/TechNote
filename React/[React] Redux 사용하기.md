## Redux란 ?
 - React에서 Flux 아키텍처를 편하게 사용할 수 있도록 해주는 라이브러리 이다.


## 의존모듈
1. redux
1. react-redux

## 설치하기
~~~
npm install --save redux react-redux
~~~
