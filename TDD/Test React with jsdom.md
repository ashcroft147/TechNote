## jsdom 이란 ?
DOM이란 web browser에서 보여지는 document의 UI가 어떤 Node Tree로 구성되는지 보여주는 객체 모델이다.

## jsdom이 test conde 실행시 필요한 이유 ?
결국에는 Browser 없이 DOM 테스트를 수행하기 위해 필요한 모듈로써 fake browser 환경을 제공
